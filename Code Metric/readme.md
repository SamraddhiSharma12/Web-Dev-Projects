# 🧮 Code Metric

Code Metric is a web application that analyzes your **LeetCode progress** 
---

## 🌟 Features

✅ Fetches your **LeetCode profile data** dynamically  
✅ Displays total problems solved with difficulty-wise split    
✅ Real-time data updates
